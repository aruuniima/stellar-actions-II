# stellar-actions-II Repository

This repository contains the entire code and part of the data for the analysis of stellar actions in a galactic simulation focussing on clustering in action space, which is part of the paper titled "*Evolution of action-space coherence in a Milky Way-like simulation*" (submitted) by Arunima et al. The project includes several directories for different aspects of the analysis, and each directory has a README file that describes the directory and its scripts in detail.
